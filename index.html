Welcome to the Island Stats CDN Docs

The Island Stats CDN is a free to use CDN for developers and people who wish to use icons from MCC Island in their projects without the need to self host them.

Here you can find all the information you need to use the Island Stats CDN. If you have any questions, please join our Discord server and ask in the #cdn channel.

Getting Started

The base URL for the CDN is https://cdn.islandstats.xyz/. You can use this to access any of the files on the CDN. Images are sorted into folders based on their type. For example, all rank icons are in the /ranks directory. So for the mod rank icon, the URL would be ranks/mod.png. You can find a list of all the directories on the CDN below.

Directories

Directory Description
/ranks Contains all the rank icons for MCC Island.
/ranks_long Contains all the rank icons for MCC Island, but with the full rank name.
/badges Contains all the badge icons for MCC Island. Each badge is split into it's own category.
/emojis Contains all the emojis for MCC Island.
/faction Contains all the faction icons for MCC Island. Each faction's prestige icons are split into their respective factions.
/passes Contains all game pass and battle pass icons for MCC Island.
/events Contains icons from events on MCC Island. Each event is split into it's own category.
/icons Contains extra icons for MCC Island, such as currency icons.

Faction Icons

Faction icons are split into their respective factions. The icon for the faction is named after it's prestige level, starting from 0. For example, the base icon for the Red Rabbits faction would be red/0.png. The icon for prestige 1 of the Blue Bats faction would be blue/1.png. The icon for prestige 2 of the Green Geckos faction would be green/2.png. And so on.

Game Passes and Battle Pass

Game pass and battle pass icons are named after their respective game or /battle_pass. For example, the icon for the Dynaball Slapstick game pass would be dynaball/slapstick.png and for the Ultimate Battle Pass it would be battle_pass/ultimate.png.

You can find a list of all the pass categories below.

  • battle_box
  • battle_pass
  • dynaball
  • hitw
  • pkw
  • sky_battle
  • tgttos
